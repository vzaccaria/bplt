Usage:
    bplt [ co | checkout ]  REPO [ -n NAME ] [ -c CONFIG ]
    bplt [ ls | list ]
    bplt -h | --help

Options:
    -n, --name NAME         name of the new project
    -c, --config CONFIG     list of comma separated, key=value pairs

Commands:
    co, checkout        checkout a repo
    ls, list            list available repos
